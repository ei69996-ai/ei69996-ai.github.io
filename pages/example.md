# 첫 번째 게시글

안녕하세요! 이것은 GitHub Pages 정적 블로그의 첫 번째 게시글입니다.

## 기능 소개

이 블로그는 다음과 같은 기능을 제공합니다:

- **마크다운 지원**: 마크다운 형식으로 게시글 작성
- **다크/라이트 모드**: 사용자 친화적인 테마 전환
- **검색 기능**: 게시글 제목, 내용, 태그 검색
- **태그 필터링**: 태그별 게시글 필터링
- **코드 하이라이팅**: Prism.js를 사용한 코드 문법 강조
- **댓글 시스템**: Giscus를 통한 GitHub Discussions 연동

## 코드 예시

다음은 JavaScript 코드 예시입니다:

```javascript
function greet(name) {
  return `안녕하세요, ${name}님!`;
}

console.log(greet('방문자'));
```
